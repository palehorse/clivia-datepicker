## clivia-datepicker
簡易的日期選擇器
## 前置安裝
    jQuery
## 安裝
1.用npm指令安裝
```sh
npm install clivia-datepicker
```
2.用html語法引入
```html
<script src="clivia-datepicker/clivia-datepicker.jquery.js"></script>
```
#### Vue
```javascript
require('clivia-datepickerr/clivia-datepicker.jquery.js');
```
## 使用方法
#### 初始化
``` javascript
$('#datepicker').datepicker();
```
